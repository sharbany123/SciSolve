Code Review: Shor's Algorithm Implementation

Submitted by Dr. Lisa Zhang | Last updated: Today, 9:30 AM

quantum_fourier_transform.qs

operation QuantumFourierTransform(register : Qubit[]) : Unit { let n = Length(register); for i in 0..n-1 { H(register[i]); for j in 1..n-i-1 { Controlled R1([register[i+j]], (2.0 * PI() / (2.0^(j+1)), register[i]); } } // Qubit reversal needed here }
Review Note: You're missing the qubit reversal step at the end of the QFT. For Shor's algorithm to work correctly, you'll need to add:
// Add after the main QFT operations for i in 0..n/2-1 { SWAP(register[i], register[n-i-1]); }

modular_exponentiation.qs

operation ModularExponentiation(a : Int, N : Int, ctrls : Qubit[], target : Qubit[]) : Unit { let L = Length(target); mutable result = 1; for idx in 0..Length(ctrls)-1 { set result = (result * PowD(a, 2^idx)) % N; (ControlledOnInt(1, ApplyToEachCA(X, _)))( [ctrls[idx]], target w/ result ); } }
Review Note: The exponentiation logic looks correct, but you might want to add intermediate measurements when scaling beyond 12 qubits to prevent phase drift. Consider adding:
if L > 12 { using (ancilla = Qubit()) { // Intermediate verification logic // ... } }
